# ยง 878.5080   Air-handling apparatus accessory.

(a) *Identification.* An air-handling apparatus accessory is a supplementary device that is intended to be used with an air-handling apparatus for a surgical operating room. This device provides an interface between the components of the device or can be used to switch electrical power. This generic type of device includes fittings, adapters, couplers, remote switches, and footswitches.


(b) *Classification.* Class I (general controls). The device is exempt from the premarket notification procedures in subpart E of part 807 of this chapter subject to the limitations in ยง 878.9.
