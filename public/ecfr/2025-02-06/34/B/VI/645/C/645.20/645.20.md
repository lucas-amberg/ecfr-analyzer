# ยง 645.20   How many applications for an Upward Bound award may an eligible applicant submit?

(a) An applicant may submit more than one application as long as each application describes a project that serves a different target area or target school, or another designated different population.


(b) For each grant competition, the Secretary designates, in the _Federal Register_ notice inviting applications and other published application materials for the competition, the different populations for which an eligible entity may submit a separate application.
