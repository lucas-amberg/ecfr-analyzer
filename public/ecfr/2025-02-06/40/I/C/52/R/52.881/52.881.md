# ยง 52.881   PM

The state has submitted a committal SIP for Kansas City, Kansas. The committal SIP contains all the requirements identified in the July 1, 1987, promulgation of the SIP requirements for PM_10_ at 52 FR 24681, except the state will report the PM_10_ data which exceed the standard within 60 days of the exceedance, rather than 45 days.
