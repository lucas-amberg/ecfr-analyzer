# ยง 93.117   Criteria and procedures: Compliance with PM

The FHWA/FTA project must comply with any PM_10_ and PM_2.5_ control measures in the applicable implementation plan. This criterion is satisfied if the project-level conformity determination contains a written commitment from the project sponsor to include in the final plans, specifications, and estimates for the project those control measures (for the purpose of limiting PM_10_ and PM_2.5_ emissions from the construction activities and/or normal use and operation associated with the project) that are contained in the applicable implementation plan.
