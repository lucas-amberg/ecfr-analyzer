# ยง 98.522   GHGs to report.

You must report:


(a) CO_2_ process emissions from each ceramics process unit (*e.g.,* kiln, dryer, or oven).


(b) CO_2_ combustion emissions from each ceramics process unit.


(c) CH_4_ and N_2_O combustion emissions from each ceramics process unit. You must calculate and report these emissions under subpart C of this part by following the requirements of subpart C of this part.


(d) CO_2_, CH_4_, and N_2_O combustion emissions from each stationary fuel combustion unit other than kilns, dryers, or ovens. You must report these emissions under subpart C of this part by following the requirements of subpart C of this part.
